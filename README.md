Template Website for Athletic Trainers & CPTs
This project is a customizable template website designed for Athletic Trainers and Certified
Personal Trainers (CPTs). It includes a modern layout and built-in Calendly integration for booking
sessions.
Features
- Responsive design
- Built for fitness professionals
- Calendly booking integration
- Easy customization
- Lightweight setup
Getting Started
1. Download the project
2. Open in Visual Studio Code
3. Install dependencies:
npm install
4. Start the development server:
npm run dev
5. Open in your browser:
http://localhost:8080
Project Structure
project/
■■■ src/
■■■ public/

Customization
Update text, images, branding, and replace the Calendly link with your own to start accepting
bookings.
